Text Editing

How to create rich format text in Notes and Comments.

Markdown

Heads Up! This page is a work in progress.

Markdown is a formatting language you can use to create rich text format on the web. Convexys supports a subset of Markdown and added some features specific to our application.

To learn more about markdown syntax, here's a quick cheatsheet.

Our Flavor

We added some symbols and tags to use only on Convexys to inhance objects linking such as users and items in your text.
These can be used in notes and stream comments.

  • Members: You can add a link to a user profile by using the syntax @[Full Name].
    Example: @[John Smith] will be replaced by John Smith pointing to John Smith profile page
  • Items: You can add a link to an item page by using the syntax i#item-number.
    Example: i#10-000 will be replaced by 10-000 pointing to the page of item number 10-000.

Authorized HTML tags

Not all HTML tags are authorized in the notes and comments. You can use only the following tags:

  • Inline tags: a, img, span, strong, i, br, b, code
  • Block tags: p, div, h1, h2, h3, h4, h5, h6, blockquote, pre, table, dl, ol, ul, address, math
