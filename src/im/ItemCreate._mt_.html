Create an item

###t/wip.html

This section details how to create a new item.

Follow these steps to create a new item:

  1. Go to the items page from the main menu or using this url: www.convexys.com/productinformation/items.
    Items link
  2. Hover on the data table and the (+) sign will appear at the top right.
    The create item button
  3. Click to display the ‘Create Item’ form.
    The create item form
  4. Fill in the form as follow:

    • Item Number: Type in a number for the item. This must be unique. It can contain only alphanumerics and dashes, no spaces. Depending on your organization’s rules you may use significant or non-significant numbers. We recommend to use non-significant incremental numbers only with dashes as separators.

      The system proposes a number based on an ascending sort of your existing item numbers. It takes the last one and it increments to generate the next number. If you do not have any item at all, the system proposes to start at 10–000.

    • Number: This is the first revision of the item to start with in the system. If you have your items already in an old software and you choose to keep the data in your legacy system and your item as at revision 03 today, then enter 03. If Your item is new, then you can start with 00 or any other format you choose. This number can contain only alphanumerics, no spaces. We recommend that you use numbers only.
    • Name: A short name for the item
    • Description: A detailed description of the item.
    • Type: The type of the item. Convexys allows you to manage different types of items from a Product Family to a Service or a Software. Learn more about our item types.
    • Unit of Measure: The default unit of measure to use to measure the item. E.g. A liquid it is very likely to be measured in Liters. A wire is usually measured in length units (M, DM, CM). If the item is a discrete item then the appropriate unit would be PC (piece). We recommend that you use metric units only.
  5. Click save to add the item.
  6. If any data is not provided as expected the system will return an error message at the top of the form.
    Error message on item create

Your item is now in.
